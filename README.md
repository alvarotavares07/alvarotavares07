## Sobre mim

Trabalho como Analista de Sistemas com um perfil que cruza análise de negócio, entrega técnica e automação. Não fico só no papel de documentar requisitos — eu entro no processo, entendo a dor real do cliente, prototipo, automatizo e acompanho em produção.

Boa parte da minha experiência está no setor bancário. Já trabalhei em projetos para **Santander, Banco Inter, Bradesco e BB Americas**, em contextos onde qualquer erro custa caro e o prazo não espera. Aprendi a ser preciso.

Tenho experiência com **RPA usando S-Works (Evertec)**, integração de **APIs REST** validadas via Postman e Swagger, extração de dados por **OCR com Atomics**, e participei de uma migração de **Redis em ambiente Java** — desde o mapeamento de impacto até o monitoramento pós-deploy.

Mais recentemente, incorporei **Devin AI e GitHub Copilot** no fluxo de trabalho — não como curiosidade, mas como acelerador real na análise de código, geração de testes e revisão de integrações.

O ciclo que costumo percorrer inteiro:

```
levantamento de requisitos  →  prototipação  →  automação
     integração  →  homologação  →  produção  →  monitoramento
```
